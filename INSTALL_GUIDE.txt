================================================================================
RATEEYE INSTALLATION GUIDE
================================================================================

Thank you for choosing RateEye! This guide will help you get the application 
running on your computer in just a few minutes.

1. HOW TO INSTALL
-----------------

MacOS (.dmg):
  1. Open the downloaded "rateeye-macos.dmg".
  2. Drag the RateEye icon to your "Applications" folder.
  3. You can now launch RateEye from your Applications folder.

Windows (.exe Installer):
  1. Run the "rateeye-setup.exe" file.
  2. Follow the on-screen prompts to select your installation folder.
  3. A shortcut will be created on your Desktop.

2. RUNNING THE APPLICATION FOR THE FIRST TIME
----------------------------------------------

MacOS:
  1. Double-click "RateEye" in your Applications folder.
  2. A message may say "Apple could not verify..." 
  3. Click "Cancel" or "OK".
  4. Go to "System Settings" > "Privacy & Security".
  5. Scroll down to the "Security" section and click "Open Anyway" 
     next to the message about RateEye.
  6. Confirm by clicking "Open" on the final popup.

Windows:
  1. Double-click the "RateEye" shortcut on your Desktop.
  2. If a blue "SmartScreen" window appears, click "More info" 
     and then "Run anyway".

3. ACCESSING THE DASHBOARD
--------------------------

RateEye is designed to open your web browser automatically once it starts. 
A command window will open to run the server; keep this window open while 
using the app.

If your browser does not open automatically, go to: http://127.0.0.1:8000

INITIAL LOGIN (ADMIN):
  For security, you must use these temporary credentials for your first login. 
  The system will immediately prompt you to change your password.

  Username: admin
  Password: adminpassword

4. IMPORTANT NOTE ON DATA
-------------------------
RateEye will create a "data" folder in its installation directory to 
store your database. Do not delete this folder, as it contains all 
your saved yield and security information.
